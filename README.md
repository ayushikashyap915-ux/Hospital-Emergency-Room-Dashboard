# Hospital-Emergency-Room-Dashboard
An interactive Hospital Emergency Room Dashboard built in Microsoft Excel to analyze patient flow, wait times, satisfaction scores, admissions, demographics, and departmental referrals. Developed using Pivot Tables, Pivot Charts, Slicers, KPI cards, and data visualization techniques to support data-driven decision-making.
## 📌 Project Overview

This project is an interactive Hospital Emergency Room Dashboard developed in Microsoft Excel to analyze patient flow, operational efficiency, and hospital performance. The dashboard transforms raw emergency room data into meaningful business insights using Excel's advanced analytics features.

The objective of this project is to demonstrate how Excel can be used as a Business Intelligence tool for healthcare analytics, enabling data-driven decision-making through interactive dashboards and KPI monitoring.

---

## 🎯 Project Objectives

- Monitor emergency room performance.
- Analyze patient volume and admission trends.
- Measure average waiting time.
- Evaluate patient satisfaction.
- Identify referral patterns across hospital departments.
- Provide interactive filtering for better analysis.
